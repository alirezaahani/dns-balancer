# dns-balancer
Multiplexing DNS Resolver with DoH responses.


Forked from: `https://github.com/anonvector/DNS-Multiplexer`



## Features: 
- Multiplexing results from multiple UDP based DNS resolvers
- Periodic Health checking
- DoH response (Security and compatibility not tested)
- Custom rules in DNSMasq style
- IPv6 blocked (because of internet situation in Iran)
